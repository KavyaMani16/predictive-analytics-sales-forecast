# Sales Forecasting using Machine Learning

## Overview
This project uses historical sales data to predict future sales using Linear Regression. It demonstrates basic predictive analytics and trend forecasting.

## What I did
- Created a historical sales dataset (2018–2023)
- Visualized sales trends using graphs
- Built a Linear Regression model
- Predicted future sales (2024–2026)
- Plotted actual and predicted values

## Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn
- Google Colab

## Outcome
The model successfully forecasts future sales based on past trends.
